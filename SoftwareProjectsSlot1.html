To improve my familiarity with and ability to use Excel and VBA coding, I created a project organizer that sorts projects, jobs, and assignments for related tasks.

This user form is a date tracker that accurately records dates and incorporates elements such as leap years and time frames into the VBA coding. Each section of Project, Job, and Task pulls data from lists that track each column in the Project, Job, and Task pages. The data in each column are brought into the current data list on the main page on each page. Financial balances and charts are automatically updated depending on each project, job, and task.
