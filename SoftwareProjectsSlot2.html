Description

During the summer before university, I had a list of books that I wanted to read through. To help me create an efficient reading order, I designed this code to organize and sort the books on my list.

This code looks through two lists of books, Audiobooks and Paper copy books. It creates two different book object arrays; in the first array, it stores the list of audiobooks; in the second array, it looks through the paper copy list and only appends the books that do not exist in the first array. It then prints both arrays into a single file.
