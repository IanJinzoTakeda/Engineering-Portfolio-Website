Description

A minor issue that I have always encountered when writing search reports is the time it takes to format them. Traditionally, multiple lines of text, varying from class to class with different numbers of sections and subsections, tediously take about 10 to 20 minutes to format correctly.

To expedite this process, I created a Python script that takes user data into the console and stores it. Then, using the Pandas library, I can write this input data into a Mircosoft Word document with the correct headers, subheaders, authors, class, etc.
